<objective>
Isolate the root cause of the Phase 1 width-ordering regression (Gap 2 of 01-VERIFICATION.md) before any fix is applied. Per the authoritative user decision, DIAGNOSE first: run the full extraction on all 4 tracks under a uniform sweep of the two suspected mechanisms (detrend order and continuity tracking), then pre-register an outcome-independent criterion for the fix so the subsequent correction is principled, not tuned.

Purpose: Provide the evidence and the pre-registered selection criterion that plan 01-07 uses to apply ONE track-independent fix, preserving the no-tuning prohibition (SPEC Prohibitions, D-14 escalation discipline, A3 rationale).
Output: A committed diagnostic script, a machine-readable sweep table, and a committed diagnosis with the pre-registered fix-selection criterion.
</objective>
